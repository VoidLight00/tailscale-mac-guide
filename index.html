Tailscale VPN + macOS 화면 공유로 안전하고 빠르게 원격 접속하는 방법을 알려드립니다.

어디서나 접속

카페, 사무실, 해외 어디서든 OK

1 맥미니 화면 공유 활성화

📍 시스템 설정 경로

  1. 1. 맥미니에서 시스템 설정 열기 (구버전은 "시스템 환경설정")
  2. 2. 일반 → 공유 클릭
  3. 3. "화면 공유" 토글 켜기 (원격 관리는 OFF 필수!)
  4. 4. 접속 허용: "이 Mac의 모든 사용자" 또는 특정 계정 선택
중요!

"원격 관리"와 "화면 공유"를 동시에 켜면 VNC 프로토콜 충돌 발생. 원격 관리는 반드시 OFF로 설정하세요.

2 Tailscale 설치 (맥미니 + 맥북 둘 다)

🍎 App Store에서 설치

  1. 1. Mac App Store에서 "Tailscale" 검색
  2. 2. 무료 다운로드 후 설치
  3. 3. 앱 실행 → 같은 계정으로 로그인 (Google/Apple/Microsoft 중 선택)
  4. 4. 상단 메뉴바에 Tailscale 아이콘 뜨면 완료 
# Homebrew로 설치하는 방법도 있음 (선택)
brew install tailscale

✅ 설치 확인

  • 상단 메뉴바에서 Tailscale 아이콘 클릭 → 기기 목록에 두 맥이 보임
  • 맥미니 이름 옆에 IP 주소 (예: 100.x.x.x)

3 맥북에서 맥미니 접속

🖥️ Finder "서버에 연결" 방법

  1. 1. 맥북에서 Finder 열기
  2. 2. 상단 메뉴 이동 → 서버에 연결… (⌘K)
  3. 3. 서버 주소 입력: 
vnc://100.x.x.x

또는 MagicDNS 사용 (더 편함):

vnc://macmini.your-tailnet.ts.net
  1. 4. 연결 클릭
  2. 5. 맥미니 계정 아이디/비밀번호 입력
  3. 6. 화면 품질: 고성능 선택 (소리/해상도 지원)
Tip

Tailscale 메뉴에서 맥미니 우클릭 → "Copy IP" 하면 주소 바로 복사됩니다.

! 문제 해결

Tailscale 목록에 기기가 안 보여요

원인:

  • 로그인 계정이 다름
  • 인터넷 연결 끊김
  • Tailscale 세션 만료 (5개월)

해결:

  • 두 맥 모두 같은 계정으로 로그인했는지 확인
  • Tailscale 앱 재로그인 시도
  • 와이파이 연결 확인 (핫스팟 추천, 아래 참고)
외부 와이파이에서 연결 안 됨

원인:

공공 와이파이/카페 네트워크가 VPN 포트 차단

해결:

🔥 핫스팟 사용 (최고의 해결책)

  1. 아이폰/안드로이드 개인 핫스팟 켜기
  2. 맥북을 핫스팟에 연결
  3. Tailscale 연결 → 안정적 터널 형성

핫스팟은 데이터 사용량 적음 (화면 공유는 네트워크 효율적)

화면이 까맣게 나오거나 연결 끊김

원인:

  • 맥미니 화면 공유 꺼짐
  • 잠자기 모드 진입
  • "원격 관리"와 충돌

해결:

  • 화면 공유 ON 재확인 (시스템 설정 → 공유)
  • 원격 관리 OFF 필수
  • 에너지 절약: "전원 어댑터 연결 시 잠자기 안 함" 설정
  • 외장 모니터 연결된 경우: 전원/디스플레이 설정 확인

고급 팁

🌙 맥미니 24시간 켜두기

외부 접속용이라면:

  • 에너지 절약 → "디스플레이 끄기" 시간 설정
  • "전원 어댑터 연결 시 자동으로 잠자기 해제"
  • 디스플레이는 꺼지지만 시스템 깨어있음

📱 아이패드/아이폰 접속

모바일에서도 가능:

  • App Store에서 Tailscale + RealVNC Viewer 설치
  • 동일한 방법으로 vnc://IP 접속
  • 터치로 마우스 제어 가능

🔐 보안 강화

  • Tailscale ACL 설정으로 특정 기기만 접근 허용
  • VNC 암호 별도 설정 (화면 공유 설정에서)
  • 2FA 활성화 (Tailscale 계정)

⚡ 속도 최적화

  • Tailscale은 가능하면 직접 P2P 연결 (DERP 릴레이 회피)
  • 화면 공유 품질: "고성능" vs "최상 품질" 선택
  • 유선랜 사용 시 더 안정적

자주 묻는 질문

Tailscale 무료인가요?
네, 개인 사용은 완전 무료입니다. 최대 100대 기기, 1명 사용자까지 무제한. 팀 플랜은 유료입니다.
포트포워딩 필요한가요?
전혀 필요 없습니다. Tailscale이 자동으로 NAT를 뚫고 P2P 연결을 만들어줍니다. 공유기 설정 변경 불필요.
Windows에서도 되나요?
네! Tailscale은 Windows, Linux, Android, iOS 모두 지원합니다. 윈도우 → 맥 접속도 가능하며, RDP 대신 VNC를 사용하면 됩니다.
속도가 느려요
  • 화면 공유 품질을 "최상 품질" → "고성능"으로 변경
  • 핫스팟 사용 (공공 와이파이보다 빠름)
  • Tailscale 상태에서 "Direct" 연결 확인 (Relay 사용 시 느림)
