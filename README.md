# 🏠 맥 원격 접속 완벽 가이드

Tailscale + macOS 화면 공유로 외부에서 집 맥미니를 안전하게 제어하는 방법.

## ✨ Features

- ✅ **10분 설정**: 복잡한 네트워크 지식 불필요
- ✅ **안전한 연결**: WireGuard 기반 암호화 터널
- ✅ **포트포워딩 불필요**: NAT 자동 우회
- ✅ **외부 와이파이 문제 해결**: 핫스팟 활용법 포함
- ✅ **다크 모드 기본**: 개발자 친화적 디자인
- ✅ **모바일 완전 반응형**: 아이패드/폰에서도 편하게

## 🚀 Quick Start

1. 이 레포 클론 또는 다운로드
2. `index.html` 열기 (로컬에서 바로 실행 가능)
3. 또는 Vercel에 배포

## 📦 Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/tailscale-guide)

또는 수동 배포:

```bash
npm i -g vercel
cd tailscale-guide
vercel
```

## 🛠️ Tech Stack

- **Vanilla HTML** + **Tailwind CSS CDN**
- **No build process** - 즉시 실행 가능
- **Lightweight** - 단일 HTML 파일

## 📝 Customization

### 색상 변경

`tailwind.config` 섹션에서 Tailscale 컬러 팔레트 수정:

```javascript
colors: {
    tailscale: {
        500: '#22c55e', // 메인 컬러
        // ...
    }
}
```

### 콘텐츠 수정

HTML 내부의 섹션별로 텍스트 직접 수정 가능.
